Smart Pension Network Login Instructions

Welcome to Smart Pension!

This application is running on your local network. Follow the instructions below to access and log in to the system.

Accessing the Application

Open your web browser (Chrome, Firefox, Edge, etc.)
Enter the following URL in your address bar: http://172.20.10.2:3000
The Smart Pension login page should appear

For Admin Users

Use the admin account provided by your system administrator
Enter your wallet address (if applicable)
For testing, you can use a test wallet address or create a new one

For Pensioners

You'll need your assigned wallet address
Contact the admin if you don't have your wallet address
Enter your wallet address in the login field
Note: This is a local network application. It is not accessible from the internet, only from devices connected to the same network.
